# Mohamad Hamdo Portfolio

Premium bilingual portfolio website for Mohamad Hamdo with React, Vite, Express API, contact storage, and a lightweight admin dashboard.

## Run

```powershell
npm install
npm run dev
```

Backend:

```powershell
copy .env.example .env
npm run server
```

Frontend dev server runs on Vite. Backend defaults to `http://localhost:4174`.

## Export ZIP

```powershell
npm run zip
```

The ZIP is created in `release/mohamad-hamdo-portfolio.zip`.

## Background Asset

The background texture lives at `public/assets/arabic-pattern.svg`. Replace it with the uploaded Arabic pattern image using the same filename path reference in `src/styles.css` if you want the exact supplied artwork.
